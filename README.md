# Ideation Process
- Export Chat GPT history and feed it into Opus 4.6 along with job posting and have it suggest ideas for a project. 
- Brainstorm idea in Plan mode to drill into idea
- Generate initial spec to provide to Figma Make to generate UI mockups to visualize the idea (I am a highly visual thinker/learner)
- Generate and tweak docs.mdc with v0.1 of the project background and scope
- Utilize the BMAD Method in brainstorm mode to drill down to a concrete project scope and requirements document using first principles thinking.
